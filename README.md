# 📧 SmartMail Assistant – AI Powered Email Reply Generator

SmartMail Assistant is a full-stack AI-powered web application that helps users generate professional email replies instantly using Google Gemini AI. Users can paste an email, select the desired tone, and receive a context-aware response generated by AI.

---

## 🚀 Features

- 🤖 Generate intelligent email replies using Gemini AI
- 📝 Paste original email content and get instant responses
- 🎭 Select different email tones (Professional, Friendly, Formal, etc.)
- ⚡ Fast API communication between React frontend and Spring Boot backend
- 🔐 Secure API key management using environment variables
- 🌐 RESTful API architecture
- 📱 Clean and responsive user interface

## ⚙️ Backend Setup

### 1. Clone Repository

```bash
git clone https://github.com/AdityaShendkar/SmartMail-Assistant.git
```

### 2. Navigate to Backend

```bash
cd backend
```

### 3. Configure Environment Variables

Add your Gemini API credentials:

#### Windows

```bash
set GEMINI_URL=https://generativelanguage.googleapis.com/v1beta/models/gemini-flash-latest:generateContent?key=
set GEMINI_KEY=YOUR_API_KEY
```

#### Linux / Mac

```bash
export GEMINI_URL=https://generativelanguage.googleapis.com/v1beta/models/gemini-flash-latest:generateContent?key=
export GEMINI_KEY=YOUR_API_KEY
```

### 4. Run Spring Boot Application

```bash
./mvnw spring-boot:run
```

Backend will start at:

```
http://localhost:8080
```

## 🔌 API Endpoint

### Generate Email Reply

**POST**

```
/api/email/generate
```

### Request Body

```json
{
  "emailContent": "Hello, can we schedule a meeting tomorrow?",
  "tone": "Professional"
}
```

### Response

```text
Hello,

Thank you for reaching out. I would be happy to meet with you tomorrow. 

I am available at the following times (please note your time zone):
* [Insert Time Option 1]

Please let me know if either of these options works for you, and I will send over a calendar invitation with the meeting details. If you have a different time in mind, please feel free to suggest it.

Best regards,

[Your Name] 
[Your Title]
```

## 🔄 Application Workflow

1. User enters the original email.
2. User selects the preferred reply tone.
3. React sends the request to Spring Boot API.
4. Backend creates an AI prompt.
5. WebClient sends the request to Gemini AI API.
6. Gemini generates an email reply.
7. The response is displayed on the frontend.
